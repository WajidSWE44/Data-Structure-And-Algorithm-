# 🚀 Data Structures & Algorithms

A collection of **Data Structures and Algorithms (DSA)** implemented in **Java**. This repository contains solutions to common algorithmic problems and implementations of fundamental data structures, making it useful for learning, interview preparation, and academic practice.

---

## 📌 About

This repository includes Java programs covering various topics such as:

- Searching Algorithms
- Sorting Algorithms
- Graph Algorithms
- Hashing Concepts
- Linked Lists
- Stacks
- Database Connectivity
- Object-Oriented Programming (OOP)
- Miscellaneous DSA Problems

The goal of this repository is to strengthen problem-solving skills and provide clean Java implementations of popular algorithms.

## ▶️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/WajidSWE44/Searching-Algorithms.git
```

### Navigate to the Project

```bash
cd Searching-Algorithms
```

### Compile

```bash
javac FileName.java
```

### Run

```bash
java FileName
```

Example:

```bash
javac BinarySearchPlusSort.java
java BinarySearchPlusSort
```
